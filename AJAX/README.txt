AJAX ASSIGNMENT

Folder structure:
AJAX/
├── ajax_basic.html
├── content.html
└── style.css

IMPORTANT:
Run this project through XAMPP/Apache.

Copy the AJAX folder into:
C:\xampp\htdocs\

Then open:
http://localhost/AJAX/ajax_basic.html

Do not open ajax_basic.html directly with file:// because XMLHttpRequest may be blocked.
