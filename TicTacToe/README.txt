TIC TAC TOE ASSIGNMENT

Open TicTacToe.html in a browser, or use the Live Server extension in Visual Studio Code.

Folder contents:
- TicTacToe.html
- css/game.css
- js/tictactoe.js
- images/x.png and images/o.png
- media/place.mp3, media/tie.mp3, and media/winGame.mp3
